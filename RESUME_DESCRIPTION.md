    Technologies: Python 3.11, Pandas, NumPy, Matplotlib, Seaborn, Plotly, Streamlit,
                  SQL, Microsoft Excel, CSV/Excel, Power BI, GitHub.

•   Built an interactive analytics dashboard to monitor restaurant KPIs — rating trends,
    price-category distribution, and city-wise performance across 15 Indian cities
    using Streamlit and Plotly with real-time filter controls

•   Developed EDA and statistical analysis workflows for cuisine popularity, trend analysis
    of online vs offline order impact, and correlation study between cost and rating
    across 1,280+ restaurant records

•   Generated 8 data visualizations including bar charts, scatter plots, donut charts,
    and correlation heatmaps; produced structured analytical reports for data storytelling
    and stakeholder communication

•   Managed structured datasets using SQL queries and Pandas — performed ETL processing,
    data wrangling, null handling, and custom KPI feature engineering
    (Value Score metric, Price Category segmentation)

## INTERVIEW TALKING POINTS:

Q: "Tell me about your project"
A: "I built a restaurant analytics dashboard for Zomato's India market.
    I analyzed 1,280+ restaurants across 15 cities using Python and SQL —
    cleaned the raw data, performed ETL processing, engineered custom KPIs
    like a Value Score metric, and created 8 visualizations. I then deployed
    a live interactive dashboard on Streamlit Cloud."

Q: "What SQL queries did you write?"
A: "I wrote 8+ SQL queries — city-wise KPI aggregations using GROUP BY,
    rating band segmentation using CASE WHEN, online order impact analysis
    using conditional COUNT, and value-score ranking using calculated columns.
    I also wrote a query to identify underserved cities for new restaurant entry."

Q: "What was your biggest challenge?"
A: "The Cuisines column had multiple values per row like 'North Indian, Chinese'.
    I had to split, explode, and count individual cuisines separately using
    Pandas string operations to get accurate popularity rankings."

Q: "What was your key business recommendation?"
A: "New restaurants should target Tier-2 cities — lower competition, growing market.
    The ₹300–₹600 price range captures 45% of market share, so affordable pricing
    maximizes volume for new entrants. Also, enabling online ordering increases
    customer engagement by 23% based on our vote analysis."
